🚢 Ship Reservation System

A modern full-stack web application built to simplify ship ticket reservations and passenger management. The system offers a seamless booking experience, allowing users to explore available routes, check schedules, reserve tickets, and manage their travel plans through an intuitive and responsive interface.
 🌟 About the Project

The Ship Reservation System was developed to digitize and streamline the traditional ship booking process. By combining a dynamic React frontend with a robust Spring Boot backend, the application provides a fast, secure, and user-friendly platform for managing maritime travel reservations.

Whether booking a new journey or managing existing reservations, users can complete their tasks efficiently with minimal effort.
